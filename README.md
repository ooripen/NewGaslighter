# מרכז מודעות לתקשורת מניפולטיבית

כלי חינוכי שנועד לעזור למשתמשים לזהות דפוסי תקשורת מניפולטיבית דרך תוכן אינפורמטיבי, דוגמאות מוסברות, ומשאבי תמיכה.

## 🎯 תכונות

- **מסך בדיקה ראשוני**: 4 שאלות הסכמה ומוכנות רגשית
- **תוכן חינוכי**: מידע על זיהוי דפוסים מניפולטיביים
- **דוגמאות מוסברות**: דיאלוגים לדוגמה עם הערות והסברים
- **משאבי תמיכה**: קישורים לקווי חירום ותמיכה מקצועית
- **ממשק עברית מלא**: תמיכה מלאה ב-RTL וטיפוגרפיה עברית

## 🚀 איך להתחיל

### התקנה מקומית

```bash
npm install
npm run dev
```

האתר יהיה זמין בכתובת: `http://localhost:5173`

### הטמעה כ-iframe

```html
<iframe
  src="YOUR_DEPLOYMENT_URL"
  style="width:100%;height:700px;border:none;"
  title="מרכז מודעות לתקשורת מניפולטיבית"
  allow="clipboard-read; clipboard-write">
</iframe>
```

## 🔐 אבטחה ופרטיות

- **אין איסוף נתונים**: כל המידע נשאר במכשיר של המשתמש
- **אין שרת backend**: כל התוכן טעון מראש
- **אין עוגיות**: משתמש ב-localStorage מקומי בלבד למעקב התקדמות
- **ניתן להטמעה**: עובד בטוח כ-iframe

## 🛠️ טכנולוגיות

- **React 19** + **TypeScript**
- **Vite** - build tool מהיר
- **Tailwind CSS 4** - styling עם תמיכה RTL
- **Shadcn UI** - קומפוננטות נגישות
- **Framer Motion** - אנימציות עדינות
- **Phosphor Icons** - אייקונים ברורים

## 📋 זרימת משתמש

1. **שלב מסך**: 4 שאלות בדיקה
   - תשובה "לא" על כל שאלה → הודעת יציאה
   - כל התשובות "כן" → מעבר לתוכן חינוכי

2. **שלב חינוך**: 3 טאבים
   - **מידע**: מידע על זיהוי גזלייטינג
   - **דוגמאות**: דיאלוגים מוסברים
